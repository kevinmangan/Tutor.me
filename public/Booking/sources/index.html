Booking Calendar PRO (jQuery Plugin)

Back End Version

Double click on a day or select 2 different days to edit them and view the changes in the Front End Version by clicking refresh.

Back End Features

  • Easy integration into your Content Management System (php examples provided; save in database or file).
  • Completely customizable from CSS.
  • Change text.
  • Change currency.
  • Display multiple months at once.
  • You can group days.
  • Set day internal notes (only for Back End version).
  • Set 4 statuses for days (Available, Booked, Special, Unavailable).
  • Set Price & Promo Price.
  • Set number of items available a day.
  • ...

Front End Version

If the calendar does not correspond with your edits from the Back End Version this means that somebody else is using the demo to.

Front End Features

  • Responsive.
  • Completely customizable from CSS.
  • Change text.
  • Change currency.
  • Display multiple months at once.
  • Display days with morning checkout.
  • Users can view infos about any day.
  • ...
